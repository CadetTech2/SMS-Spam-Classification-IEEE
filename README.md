# SMS-Spam-Classification-IEEE
# 📩 SMS Spam Classification using Ensemble & Deep Learning

This repository contains the research work for our paper:

**"An Ensemble and Deep Learning Based Approach for SMS Spam Classification"**

📍 Published in IEEE  
📍 Conference: ICECT 2025 (Islamabad, Pakistan)

---

## 📖 Abstract
This work presents a hybrid approach combining ensemble learning and deep learning techniques for SMS spam detection. The model achieves up to **99% accuracy**, demonstrating strong performance in handling linguistic ambiguity.
